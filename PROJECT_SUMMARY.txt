Claude-generated news articles.

Firebase config is embedded in src/lib/firebase.js - no additional env vars needed.

================================================================================
TROUBLESHOOTING
================================================================================

Common Issues:

"Cannot find module"
  → npm install

"Firebase error"  
  → Check src/lib/firebase.js config
  → Check Firebase project exists
  → Check allowlisted domains in Firebase

"Claude API error"
  → Check .env.local has ANTHROPIC_API_KEY
  → Check API key is valid at console.anthropic.com
  → Check API key has credits

"Page shows loading forever"
  → Open browser console (F12)
  → Check for JavaScript errors
  → Check network tab for failed requests

See SETUP_GUIDE.md for more troubleshooting help.
